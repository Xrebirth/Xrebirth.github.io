Arrays工具集

  • Arrays是cun公司提供的一个工具集
  • 该工具主要针对的是数组的操作
  • 例如：排序/二分等…

示例代码：

        int[] a = {12,3,85,67,2};
        // 排序
        Arrays.sort(a);
        for (int i = 0; i < a.length-1; i++) {
            System.out.print(a[i]+"  ");
        }
        System.out.println(" ");
        // 二分查找
        int index = Arrays.binarySearch(a,12);
        System.out.println("12的下标为："+index);
